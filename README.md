# Lacost
Ce repository sert de model à notre projet de groupe

Dans votre terminale tapez la commande pip install django-environ

Creez un fichier que vous nomerez .env,vous y mettrez les information suivante


DB_NAME='le nonm de votre base de donnée postgresql'
DB_USER='le nonm de votre utilisateur postgresql'
DB_PASSWORD='votre mot de passe'
DB_HOST=localhost
DB_PORT='le port de votre postgresql,par defaut c'est le 5432'
